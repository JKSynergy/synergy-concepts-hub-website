# Academy Photos Organization Guide

This folder structure is designed to organize your Synergy Multimedia Academy photos for the website gallery.

## Folder Structure & Purpose:

### 📚 **classrooms/**
- Photos of your academy classrooms
- Learning environments
- Desks, boards, presentation setups
- Modern classroom facilities

### 👥 **students-working/**
- Students actively learning
- Hands-on project work
- Group collaborations
- Students using equipment

### 🎥 **equipment/**
- Professional cameras and gear
- Computers and software setups
- Photography lighting equipment
- Video production tools

### 📸 **photography-classes/**
- Photography lessons in action
- Students learning camera techniques
- Photo shoots and setups
- Photography workshops

### 🎬 **video-production/**
- Video production classes
- Students working with video equipment
- Behind-the-scenes footage
- Video editing sessions

### 🎓 **graduations/**
- Graduation ceremonies
- Certificate presentations
- Student achievements
- Success celebrations

### 🏢 **facilities/**
- Academy building exterior/interior
- Reception areas
- Common spaces
- Academy branding displays

### 👨‍🏫 **instructors/**
- Professional photos of instructors
- Instructors teaching
- Instructor profiles
- Academy team photos

## Usage Notes:
- Use high-quality images (recommended: 1200px+ width)
- JPG or PNG formats work best
- Keep file names descriptive (e.g., "classroom-setup-01.jpg")
- Images will be automatically optimized for web display

## Current Gallery Integration:
The academy gallery section on your website will pull from these folders to showcase:
1. Modern Classroom → classrooms/
2. Hands-On Learning → students-working/
3. Pro Equipment → equipment/
4. Photography Class → photography-classes/
5. Video Production → video-production/
6. Student Success → graduations/

You can update the image paths in the HTML file to point to your local images once you've added them to these folders.
