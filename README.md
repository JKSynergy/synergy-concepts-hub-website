# Synergy Concepts Hub Website

## 🌐 Live Website
**URL:** https://synergyconceptshub.com

## 📋 Project Description
Professional website for Synergy Concepts Hub - A creative and digital transformation company based in Uganda, offering multimedia services, web development, and creative training through Synergy Multimedia Academy.

## 🎯 Services Offered
- **Photography & Videography**
- **Graphic Design & Branding**
- **Web Development & Systems**
- **Digital Marketing**
- **Multimedia Training Academy**

## 🏗️ Technical Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6)
- **Styling:** Tailwind CSS
- **Images:** Cloudinary CDN
- **Hosting:** Cloudflare Pages
- **Domain:** synergyconceptshub.com

## 📁 File Structure
```
├── index.html                 # Homepage
├── SCH_WEB_ WEBSITE.html     # Main website file
├── css/
│   └── critical.css          # Critical styles
├── js/
│   ├── main.js              # Core functionality
│   ├── main-portfolio.js    # Portfolio management
│   └── performance-monitor.js
├── images/                   # Image assets
├── manifest.json            # PWA manifest
└── sw.js                    # Service worker

```

## 🔄 Development Workflow
1. Make changes locally
2. Test with local server: `python dev-server.py`
3. Commit changes: `git add . && git commit -m "description"`
4. Push to GitHub: `git push`
5. Cloudflare Pages automatically deploys
